SpringBoot-集成websocket

本文字数： 4.7k 阅读时长 ≈ 4 分钟

前言

HTTP 协议有一个缺陷：通信只能由客户端发起

HTML5定义了WebSocket协议，能更好的节省服务器资源和带宽，并且能够更实时地进行通讯。服务器可以主动向客户端推送信息，客户端也可以主动向服务器发送信息，是真正的双向平等对话，属于服务器推送技术的一种。可以替代长轮询，用于对实时性要求比较高的场景:

  • 聊天室

  • 服务端消息实时推送

  • ……
