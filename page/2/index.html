и§ЈйҮҠеҷЁжЁЎејҸжҳҜдёҖз§Қй’ҲеҜ№зү№е®ҡй—®йўҳи®ҫи®Ўзҡ„дёҖз§Қи§ЈеҶіж–№жЎҲпјҢ дёҖиҲ¬дё»иҰҒеә”з”ЁеңЁOOPејҖеҸ‘дёӯзҡ„зј–иҜ‘еҷЁзҡ„ејҖеҸ‘дёӯ

  • жҠҠжӯЈеҲҷиЎЁиҫҫејҸи§ЈжһҗдёәиҜӯжі•ж ‘пјҢз„¶еҗҺеҶҚеҢ№й…ҚжҢҮе®ҡзҡ„еӯ—з¬ҰдёІпјҢе°ұйңҖиҰҒдёҖдёӘи§ЈйҮҠеҷЁ
  • еҪ“жҲ‘д»¬дҪҝз”ЁJDBCж—¶пјҢжү§иЎҢзҡ„SQLиҜӯеҸҘиҷҪз„¶жҳҜеӯ—з¬ҰдёІпјҢдҪҶжңҖз»ҲйңҖиҰҒж•°жҚ®еә“жңҚеҠЎеҷЁзҡ„SQLи§ЈйҮҠеҷЁжқҘжҠҠSQLвҖңзҝ»иҜ‘вҖқжҲҗж•°жҚ®еә“жңҚеҠЎеҷЁиғҪжү§иЎҢзҡ„д»Јз ҒпјҢиҝҷдёӘжү§иЎҢеј•ж“Һд№ҹйқһеёёеӨҚжқӮпјҢдҪҶеҜ№дәҺдҪҝз”ЁиҖ…жқҘиҜҙпјҢд»…д»…йңҖиҰҒеҶҷеҮәSQLеӯ—з¬ҰдёІеҚіеҸҜ

жң¬ж–Үеӯ—ж•°пјҡ 3.4k йҳ…иҜ»ж—¶й•ҝ ≈ 3 еҲҶй’ҹ

еүҚиЁҖ

дёӯд»ӢиҖ…жЁЎејҸ(Mediator): з”ЁдёҖдёӘдёӯд»ӢеҜ№иұЎжқҘе°ҒиЈ…дёҖзі»еҲ—зҡ„еҜ№иұЎдәӨдә’гҖӮдёӯд»ӢиҖ…дҪҝеҗ„дёӘеҜ№иұЎдёҚйңҖиҰҒжҳҫејҸең°зӣёдә’еј•з”ЁпјҢд»ҺиҖҢдҪҝе…¶иҖҰеҗҲжқҫж•ЈпјҢиҖҢдё”еҸҜд»ҘзӢ¬з«Ӣең°ж”№еҸҳе®ғд»¬д№Ӣй—ҙзҡ„дәӨдә’гҖӮ

дёӯд»ӢжЁЎејҸжҳҜйҖҡиҝҮеј•е…ҘдёҖдёӘдёӯд»ӢеҜ№иұЎпјҢжҠҠеӨҡиҫ№е…ізі»еҸҳжҲҗеӨҡдёӘеҸҢиҫ№е…ізі»пјҢд»ҺиҖҢз®ҖеҢ–зі»з»ҹз»„д»¶зҡ„дәӨдә’иҖҰеҗҲеәҰгҖӮ

дёӯд»ӢиҖ…жЁЎејҸд№ҹжҳҜз”ЁжқҘйҷҚдҪҺзұ»зұ»д№Ӣй—ҙзҡ„иҖҰеҗҲзҡ„пјҢеӣ дёәеҰӮжһңзұ»зұ»д№Ӣй—ҙжңүдҫқиө–е…ізі»зҡ„иҜқпјҢдёҚеҲ©дәҺеҠҹиғҪзҡ„жӢ“еұ•е’Ңз»ҙжҠӨпјҢеӣ дёәеҸӘиҰҒдҝ®ж”№дёҖдёӘеҜ№иұЎпјҢе…¶е®ғе…іиҒ”зҡ„еҜ№иұЎйғҪеҫ—иҝӣиЎҢдҝ®ж”№гҖӮеҰӮжһңдҪҝз”Ёдёӯд»ӢиҖ…жЁЎејҸпјҢеҸӘйңҖе…іеҝғе’ҢMediatorзұ»зҡ„е…ізі»пјҢе…·дҪ“зұ»зұ»д№Ӣй—ҙзҡ„е…ізі»еҸҠи°ғеәҰдәӨз»ҷMediatorе°ұиЎҢпјҢиҝҷжңүзӮ№еғҸspringе®№еҷЁзҡ„дҪңз”ЁгҖӮ

MediatorжЁЎејҸз»Ҹеёёз”ЁеңЁжңүдј—еӨҡдәӨдә’з»„д»¶зҡ„UIдёҠгҖӮдёәдәҶз®ҖеҢ–UIзЁӢеәҸпјҢMVCжЁЎејҸд»ҘеҸҠMVVMжЁЎејҸйғҪеҸҜд»ҘзңӢдҪңжҳҜMediatorжЁЎејҸзҡ„жү©еұ•гҖӮ

  • еҗ„дёӘUIз»„д»¶дә’дёҚеј•з”ЁпјҢиҝҷж ·е°ұеҮҸе°‘дәҶз»„д»¶д№Ӣй—ҙзҡ„иҖҰеҗҲе…ізі»пјӣ

  • Mediatorз”ЁдәҺеҪ“дёҖдёӘз»„д»¶еҸ‘з”ҹзҠ¶жҖҒеҸҳеҢ–ж—¶пјҢж №жҚ®еҪ“еүҚжүҖжңүз»„д»¶зҡ„зҠ¶жҖҒеҶіе®ҡжӣҙж–°жҹҗдәӣз»„д»¶пјӣ

  • еҰӮжһңж–°еўһдёҖдёӘUIз»„д»¶пјҢжҲ‘д»¬еҸӘйңҖиҰҒдҝ®ж”№Mediatorжӣҙж–°зҠ¶жҖҒзҡ„йҖ»иҫ‘пјҢзҺ°жңүзҡ„е…¶д»–UIз»„д»¶д»Јз ҒдёҚеҸҳгҖӮ

и®ҝй—®иҖ…жЁЎејҸзҡ„ж ёеҝғжҖқжғіжҳҜдёәдәҶи®ҝй—®жҜ”иҫғеӨҚжқӮзҡ„ж•°жҚ®з»“жһ„пјҢдёҚеҺ»ж”№еҸҳж•°жҚ®з»“жһ„пјҢиҖҢжҳҜжҠҠеҜ№ж•°жҚ®зҡ„ж“ҚдҪңжҠҪиұЎеҮәжқҘпјҢеңЁвҖңи®ҝй—®вҖқзҡ„иҝҮзЁӢдёӯд»Ҙеӣһи°ғеҪўејҸеңЁи®ҝй—®иҖ…дёӯеӨ„зҗҶж“ҚдҪңйҖ»иҫ‘гҖӮеҰӮжһңиҰҒж–°еўһдёҖз»„ж“ҚдҪңпјҢйӮЈд№ҲеҸӘйңҖиҰҒеўһеҠ дёҖдёӘж–°зҡ„и®ҝй—®иҖ…гҖӮ

и®ҝй—®иҖ…жЁЎејҸйҖӮз”ЁдәҺж•°жҚ®з»“жһ„зӣёеҜ№зЁіе®ҡзҡ„зі»з»ҹпјҢжҠҠж•°жҚ®з»“жһ„е’Ңз®—жі•и§ЈиҖҰ

еүҚиЁҖ

е‘Ҫд»ӨжЁЎејҸ(Command): жҠҠдёҖдёӘиҜ·жұӮжҲ–иҖ…ж“ҚдҪңе°ҒиЈ…еҲ°дёҖдёӘеҜ№иұЎдёӯгҖӮе‘Ҫд»ӨжЁЎејҸе…Ғи®ёзі»з»ҹдҪҝз”ЁдёҚеҗҢзҡ„иҜ·жұӮжҠҠе®ўжҲ·з«ҜеҸӮж•°еҢ–пјҢеҜ№иҜ·жұӮжҺ’йҳҹжҲ–иҖ…и®°еҪ•иҜ·жұӮж—Ҙеҝ—пјҢеҸҜд»ҘжҸҗдҫӣе‘Ҫд»Өзҡ„ж’Өй”Җе’ҢжҒўеӨҚеҠҹиғҪгҖӮ

е‘Ҫд»ӨжЁЎејҸжҳҜеҜ№е‘Ҫд»Өзҡ„е°ҒиЈ…гҖӮе‘Ҫд»ӨжЁЎејҸжҠҠеҸ‘еҮәе‘Ҫд»Өзҡ„иҙЈд»»е’Ңжү§иЎҢе‘Ҫд»Өзҡ„иҙЈд»»еҲҶеүІејҖпјҢе§”жҙҫз»ҷдёҚеҗҢзҡ„еҜ№иұЎгҖӮ

жҜҸдёҖдёӘе‘Ҫд»ӨйғҪжҳҜдёҖдёӘж“ҚдҪңпјҡиҜ·жұӮзҡ„дёҖж–№еҸ‘еҮәиҜ·жұӮиҰҒжұӮжү§иЎҢдёҖдёӘж“ҚдҪңпјӣжҺҘж”¶зҡ„дёҖж–№ж”¶еҲ°иҜ·жұӮпјҢе№¶жү§иЎҢж“ҚдҪңгҖӮе‘Ҫд»ӨжЁЎејҸе…Ғи®ёиҜ·жұӮзҡ„дёҖж–№е’ҢжҺҘж”¶зҡ„дёҖж–№зӢ¬з«ӢејҖжқҘпјҢдҪҝеҫ—иҜ·жұӮзҡ„дёҖж–№дёҚеҝ…зҹҘйҒ“жҺҘж”¶иҜ·жұӮзҡ„дёҖж–№зҡ„жҺҘеҸЈпјҢжӣҙдёҚеҝ…зҹҘйҒ“иҜ·жұӮжҳҜжҖҺд№Ҳиў«жҺҘж”¶пјҢд»ҘеҸҠж“ҚдҪңжҳҜеҗҰиў«жү§иЎҢгҖҒдҪ•ж—¶иў«жү§иЎҢпјҢд»ҘеҸҠжҳҜжҖҺд№Ҳиў«жү§иЎҢзҡ„гҖӮ

е‘Ҫд»Өе…Ғи®ёиҜ·жұӮзҡ„дёҖж–№е’ҢжҺҘж”¶иҜ·жұӮзҡ„дёҖж–№иғҪеӨҹзӢ¬з«Ӣжј”еҢ–пјҢд»ҺиҖҢе…·жңүд»ҘдёӢзҡ„дјҳзӮ№пјҡ

  • е‘Ҫд»ӨжЁЎејҸдҪҝж–°зҡ„е‘Ҫд»ӨеҫҲе®№жҳ“ең°иў«еҠ е…ҘеҲ°зі»з»ҹйҮҢ
  • е…Ғи®ёжҺҘж”¶иҜ·жұӮзҡ„дёҖж–№еҶіе®ҡжҳҜеҗҰиҰҒеҗҰеҶіиҜ·жұӮ
  • иғҪиҫғе®№жҳ“ең°и®ҫи®ЎдёҖдёӘе‘Ҫд»ӨйҳҹеҲ—
  • еҸҜд»Ҙе®№жҳ“ең°е®һзҺ°еҜ№иҜ·жұӮзҡ„ж’Өй”Җе’ҢжҒўеӨҚ
  • еңЁйңҖиҰҒзҡ„жғ…еҶөдёӢпјҢеҸҜд»Ҙиҫғе®№жҳ“ең°е°Ҷе‘Ҫд»Өи®°е…Ҙж—Ҙеҝ—

е‘Ҫд»ӨжЁЎејҸж¶үеҸҠеҲ°дә”дёӘи§’иүІпјҢе®ғд»¬еҲҶеҲ«жҳҜпјҡ

  • е®ўжҲ·з«Ҝ(Client)и§’иүІпјҡеҲӣе»әдёҖдёӘе…·дҪ“е‘Ҫд»Ө(ConcreteCommand)еҜ№иұЎе№¶зЎ®е®ҡе…¶жҺҘж”¶иҖ…гҖӮ

  • е‘Ҫд»Ө(Command)и§’иүІпјҡеЈ°жҳҺдәҶдёҖдёӘз»ҷжүҖжңүе…·дҪ“е‘Ҫд»Өзұ»зҡ„жҠҪиұЎжҺҘеҸЈгҖӮ

  • е…·дҪ“е‘Ҫд»Ө(ConcreteCommand)и§’иүІпјҡе®ҡд№үдёҖдёӘжҺҘж”¶иҖ…е’ҢиЎҢдёәд№Ӣй—ҙзҡ„ејұиҖҰеҗҲпјӣе®һзҺ°execute()ж–№жі•пјҢиҙҹиҙЈи°ғз”ЁжҺҘж”¶иҖ…зҡ„зӣёеә”ж“ҚдҪңгҖӮexecute()ж–№жі•йҖҡеёёеҸ«еҒҡжү§иЎҢж–№жі•гҖӮ

  • иҜ·жұӮиҖ…(Invoker)и§’иүІпјҡиҙҹиҙЈи°ғз”Ёе‘Ҫд»ӨеҜ№иұЎжү§иЎҢиҜ·жұӮпјҢзӣёе…ізҡ„ж–№жі•еҸ«еҒҡиЎҢеҠЁж–№жі•гҖӮ

  • жҺҘж”¶иҖ…(Receiver)и§’иүІпјҡиҙҹиҙЈе…·дҪ“е®һж–Ҫе’Ңжү§иЎҢдёҖдёӘиҜ·жұӮгҖӮд»»дҪ•дёҖдёӘзұ»йғҪеҸҜд»ҘжҲҗдёәжҺҘж”¶иҖ…пјҢе®һж–Ҫе’Ңжү§иЎҢиҜ·жұӮзҡ„ж–№жі•еҸ«еҒҡиЎҢеҠЁж–№жі•гҖӮ

    йҳ…иҜ»е…Ёж–Ү »

javaи®ҫи®ЎжЁЎејҸ-и§ӮеҜҹиҖ…жЁЎејҸ

еҸ‘иЎЁдәҺ 2020-11-03 жӣҙж–°дәҺ 2021-01-29 еҲҶзұ»дәҺ java пјҢ и®ҫи®ЎжЁЎејҸ
жң¬ж–Үеӯ—ж•°пјҡ 7.5k йҳ…иҜ»ж—¶й•ҝ ≈ 7 еҲҶй’ҹ

еүҚиЁҖ

е®ҡд№үеҜ№иұЎй—ҙзҡ„дёҖз§ҚдёҖеҜ№еӨҡзҡ„дҫқиө–е…ізі»пјҢеҪ“дёҖдёӘеҜ№иұЎзҡ„зҠ¶жҖҒеҸ‘з”ҹж”№еҸҳж—¶пјҢжүҖжңүдҫқиө–дәҺе®ғзҡ„еҜ№иұЎйғҪеҫ—еҲ°йҖҡзҹҘе№¶иў«иҮӘеҠЁжӣҙж–°гҖӮ

и§ӮеҜҹиҖ…жЁЎејҸ(Observer): еҸҲз§°еҸ‘еёғ-и®ўйҳ…жЁЎејҸпјҲPublish-SubscribeпјҡPub/SubпјүгҖӮе®ғжҳҜдёҖз§ҚйҖҡзҹҘжңәеҲ¶пјҢи®©еҸ‘йҖҒйҖҡзҹҘзҡ„дёҖж–№пјҲиў«и§ӮеҜҹж–№пјүе’ҢжҺҘж”¶йҖҡзҹҘзҡ„дёҖж–№пјҲи§ӮеҜҹиҖ…пјүиғҪеҪјжӯӨеҲҶзҰ»пјҢдә’дёҚеҪұе“ҚгҖӮ

Javaж ҮеҮҶеә“иҷҪз„¶жҸҗдҫӣдәҶjava.util.Observerе’Ңjava.util.ObservableиҝҷдёӨдёӘзұ»з”ЁдәҺе®һзҺ°и§ӮеҜҹиҖ…жЁЎејҸпјҢдҪҶжҳҜJava9ејҖе§Ӣе·Із»Ҹеәҹејғjava.util.Observerе’Ңjava.util.ObservableиҝҷдёӨдёӘзұ», е®һзҺ°и§ӮеҜҹиҖ…жЁЎејҸзҡ„ж—¶еҖҷдёҚжҺЁиҚҗдҪҝз”Ё:

жӯӨзұ»е’ҢObserverжҺҘеҸЈе·Іиў«ејғз”ЁгҖӮ Observerе’ҢObservableж”ҜжҢҒзҡ„дәӢд»¶жЁЎеһӢйқһеёёжңүйҷҗпјҢObservableдј йҖ’зҡ„йҖҡзҹҘйЎәеәҸжңӘжҢҮе®ҡпјҢе№¶дё”зҠ¶жҖҒжӣҙж”№дёҺйҖҡзҹҘдёҚдёҖдёҖеҜ№еә”гҖӮ еҜ№дәҺжӣҙдё°еҜҢзҡ„дәӢд»¶жЁЎеһӢпјҢиҜ·иҖғиҷ‘дҪҝз”Ёjava.beansеҢ…гҖӮ дёәдәҶеңЁзәҝзЁӢд№Ӣй—ҙиҝӣиЎҢеҸҜйқ дё”жңүеәҸзҡ„ж¶ҲжҒҜдј йҖ’пјҢиҜ·иҖғиҷ‘дҪҝз”Ёjava.util.concurrentеҢ…дёӯзҡ„е№¶еҸ‘ж•°жҚ®з»“жһ„д№ӢдёҖгҖӮ жңүе…іеҸҚеә”ејҸжөҒж ·ејҸзҡ„зј–зЁӢпјҢиҜ·еҸӮйҳ…Flow APIгҖӮ
