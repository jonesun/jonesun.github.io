概念

ELK：Elasticsearch、Logstash、Kibana,组合起来可以搭建线上日志系统

各服务作用

  • Elasticsearch:用于存储收集到的日志信息；
  • Logstash:用于收集日志，SpringBoot应用整合了Logstash以后会把日志发送给Logstash,- Logstash再把日志转发给Elasticsearch；
  • Kibana:通过Web端的可视化界面来查看日志
